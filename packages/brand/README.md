# `@kerno/brand`

This package is Kerno's canonical visual-token and asset source. Applications import `@kerno/brand/tokens.css`; TypeScript consumers import the palette from `@kerno/brand`. Plugin, documentation, and public asset copies are generated from `assets/` by `npm run brand:assets` and verified by `npm run audit:brand`.

Raw color values belong only in this package and generated brand assets. Product components consume semantic aliases such as `--color-background`, `--color-brand-primary`, `--color-success`, and `--chart-kerno`.

The Context Core uses two open faceted repository shells, a solid oxide verified core, and a detached oxide segment for extracted context. The `Kerno` wordmark uses normal positive tracking so `r` and `n` remain distinct at small sizes. Do not reintroduce negative tracking to the public lockup.
